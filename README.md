# ASN.1/GDMO editing mode for Emacs

This is a major mode for editing ASN.1 specification.
For installation, please add the following lines to your ~/.emacs:

```el
(add-to-list 'auto-mode-alist '("\\.asn1$" . asn1-mode))
```
